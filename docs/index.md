## 为什么选择素言？

> “在这个信息过载的时代，我们只想找回那个**即开即用、干干净净**的输入框。”

### ⚡️ 核心特性

- **跨平台统一体验**：Windows 与 macOS 完美复刻，无论在哪里办公，手感始终如一。
- **临时英文模式**：无需切换输入法，直接在中文模式下流畅输入英文单词，代码/变量名输入不再断流。
- **自定义皮肤系统**：支持明暗主题自动切换，更支持完全自定义。不仅好用，更要好看。
- **灵活布局切换**：同时支持 **横排** 与 **竖排** 布局，适应不同阅读习惯与屏幕空间。
- **超大候选视野**：支持扩展候选框，一次最多展示 **5x9=45** 个候选词。支持翻页查找，生僻词不再难找。
