# 站在巨人的肩膀上

素言 (SuYan) 并非从零创造，而是基于开源社区最优秀的输入法生态构建而成。如果把素言比作一辆高性能跑车，那么：

## 🏎️ 发动机：RIME (中州韵)

**[RIME | 中州韵输入法引擎](https://rime.im/)** 是素言的核心驱动力。

- **极度灵活**：它不仅仅是一个输入法，更是一个输入法算法框架。支持拼音、五笔、注音、仓颉、甚至方言（粤语、吴语等）。
- **跨平台**：RIME 的核心代码运行在 Windows、macOS、Linux 甚至 Android/iOS 上，保证了核心算法的一致性。
- **隐私**：从设计之初就坚持“数据不上云”，所有用户习惯只存储在本地。

> 💡 **素言做了什么？**
> 素言集成了 librime 核心库，处理底层的按键响应、拼音匹配和候选词排序，确保了输入的精准与稳定。

## ⛽️ 燃料与调校：雾凇拼音

**[Rime-Ice | 雾凇拼音](https://github.com/iDvel/rime-ice)** 是素言的灵魂配置。

RIME 原版虽然强大，但默认配置对新手来说过于“硬核”。雾凇拼音是一套**专为简体中文用户打造的配置方案**。

- **长期维护的词库**：包含最新的网络热词、常用短语，且剔除了大量无用的生僻词干扰。
- **顺手的逻辑**：优化了 `v` 模式（v101 查日期）、以词定字（`[` `]`）、自动纠错（`ignx` -> `ying`）等细节体验。
- **纯净**：没有广告推广词，没有奇怪的联想。

> 💡 **素言做了什么？**
> 素言内置了完整的雾凇拼音配置，并针对 Windows/macOS 的系统习惯进行了微调，让你安装即用，无需手动折腾配置文件。

## 🎨 车身与内饰：素言

**素言 (SuYan)** 是将上述一切整合并交付给你的最终产品。

- **现代化 UI**：重写了 Windows (TSF) 和 macOS 的候选窗口，支持高分屏、皮肤系统和明暗主题。
- **可定制词库**：内置便捷的词库管理功能，支持导入 Rime 格式词库与自定义短语，让专业术语输入得心应手。
- **生产力工具**：在输入法之外，集成了剪贴板历史、截图、取色等高频效率工具。
- **安装与分发**：提供了一键安装包、自动更新机制和友好的设置界面。
