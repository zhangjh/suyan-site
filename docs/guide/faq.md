# 常见问题

## 安装与安全

### 为什么 Windows 会提示“未知发布者”或杀毒软件拦截？
素言是由个人开发者维护的开源项目，未购买昂贵的 Windows 数字签名证书（每年需数千元）。因此：
- 安装时可能会弹出 SmartScreen 蓝屏拦截，请点击“更多信息” -> “仍要运行”。
- 部分杀毒软件可能会误报。请放心，素言代码完全开源，绝无任何恶意行为。建议将安装目录加入白名单。

### macOS 安装后无法打字？
macOS 权限管理较严。首次安装后：
1. 请确保已在“系统设置 -> 键盘 -> 输入法”中添加了素言。
2. 尝试注销或重启系统（输入法必须在重启后才能完全加载）。
3. 确保在“系统设置 -> 隐私与安全性 -> 辅助功能”中授予了素言权限（如果有提示）。

## 功能使用

### 如何切换横排/竖排？
支持在横排与竖排之间快速切换：
- **Windows**: `Ctrl + Shift + L`
- **macOS**: `Command + Shift + L`

你也可以在设置菜单中更改此行为。

### 候选词太少了，我想多看几个？
素言默认支持 **5x9** 超大候选词视图。当你翻页时，候选窗口会自动展开：
- **横排模式**：按 `↓` (向下键) 翻页。
- **竖排模式**：按 `→` (向右键) 翻页。

### 如何导入我以前的搜狗/百度词库？
素言基于 RIME 引擎（雾凇拼音配置），**不支持直接导入**搜狗或百度的加密词库文件（如 .scel）。
建议使用 [深蓝词库转换](https://github.com/studyzy/imewlconverter) 等工具，将这些词库转换为 **RIME 格式** 的文本文件，然后通过素言的“词库管理”功能导入。

### 为什么有时候截图快捷键不灵？
请检查是否与其他软件（如微信、QQ）的截图快捷键冲突。建议在素言设置中修改截图快捷键，或者关闭其他软件的全局热键。

## 隐私

### 素言真的完全不联网吗？
是的。除了检查更新（如果你手动点击了“检查更新”）外，素言的输入核心进程从不发起任何网络请求。你的每一次击键、每一个剪贴板记录都只保存在本地硬盘上。
