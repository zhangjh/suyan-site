# 雾凇拼音 (Rime-Ice)

素言的灵魂——词库与输入逻辑，完全基于开源界备受赞誉的 **[雾凇拼音 (Rime-Ice)](https://github.com/iDvel/rime-ice)** 项目。

## 为什么选择雾凇拼音？

> Rime 配置：雾凇拼音 | 长期维护的简体词库

### 🌿 干净纯粹
它没有那些乱七八糟的网络热词污染，保留了中文输入的纯粹感。词库由社区长期精心维护，确保常用词准确，生僻词不缺失。

### 🧠 智能纠错
不仅词库好，它的逻辑配置也非常出色。
- **自动纠错**：`ignx` -> `ying`，`shign` -> `shing`，打快了也不怕。
- **中英混输**：直接输入 `vsc` 出 `Visual Studio Code`，输入 `py` 出 `Python`。
- **以词定字**：按 `[` 取第一个字，按 `]` 取第二个字。
